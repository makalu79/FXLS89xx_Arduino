enum ref_updm
Enumerator
FXLS89xx_UPDATED_REF 

Reference value is updated when SDCD_OT_EA flag transitions from false to true (SDCD interruption occurs). Reference+/-threshold is compared with acceleration value.

FXLS89xx_FIXED_REF 

Reference value is fixed when SDCD function is enabled or REF_UPD bit is set. Reference+/-threshold is compared with acceleration value.

FXLS89xx_DIFF_PREVIOUS 

Delta between current and previous value is compared with threshold.

FXLS89xx_ABSOLUTE 

Reference is fixed and absolute acceleration value is compared with threshold.
